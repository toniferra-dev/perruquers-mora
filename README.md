# Perruquers Mora - Sitio Web

Sitio web oficial de Perruquers Mora, peluquería en Palma de Mallorca con tres generaciones de experiencia.

## Descripción

Sitio web elegante y profesional que refleja la calidad y el compromiso de Perruquers Mora con productos naturales AVEDA.

## Características

- Diseño responsive y moderno
- 6 páginas HTML:
  - Inicio
  - Salón
  - Servicios
  - Política de Cookies
  - Condiciones de Uso
  - Avisos Legales
- Paleta de colores natural y sofisticada
- Integración con productos AVEDA
- Optimizado para SEO

## Tecnologías

- HTML5
- CSS3 (con variables CSS)
- Google Fonts (Playfair Display & Lato)
- Design System personalizado

## Deploy en Vercel

### Opción 1: Deploy desde la interfaz web

1. Ve a [vercel.com](https://vercel.com)
2. Crea una cuenta o inicia sesión
3. Click en "Add New Project"
4. Importa este repositorio
5. Vercel detectará automáticamente que es un sitio estático
6. Click en "Deploy"

### Opción 2: Deploy desde CLI

```bash
# Instala Vercel CLI
npm i -g vercel

# Navega al directorio del proyecto
cd /Users/toniferra/Programacion/2026/Perruqueria

# Deploy
vercel
```
